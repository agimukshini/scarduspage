## Logo Requirements

- Format: PNG or SVG (preferred for scalability)
- Size: At least 200x200 pixels for high-resolution display
- Background: Transparent background preferred
- File naming: Use the exact filenames specified above

## Legal Considerations

- Ensure you have permission to use each partner's logo
- Check each partner's brand guidelines for proper usage
- Only use logos after receiving explicit permission from each partner
- Include appropriate attribution if required by the partner
